CLI for searching and viewing devdocs.io

Usage
-----

Install
------------

Install via pip
^^^^^^^^^^^^^^^

::

    pip install devdocs-cli

Install via source
^^^^^^^^^^^^^^^^^^

extract the tarball and run::

    python setup.py install
